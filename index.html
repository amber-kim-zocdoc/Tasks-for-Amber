🔗 Connect to GitHub

To save your changes so everyone sees the same dashboard, you need a GitHub Personal Access Token. This is a one-time setup — takes about 2 minutes.

  1. Go to github.com/settings/tokens/new
  2. Name it: Amber Dashboard
  3. Expiration: No expiration
  4. Check the repo scope (first checkbox)
  5. Click Generate token
  6. Copy the token and paste it below

🔒 Your token is stored only in your browser's localStorage and is never shared or transmitted anywhere except directly to GitHub.
